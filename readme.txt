This project was made under MVC patter and POO php.

To run the project just unzip the file and move to folder called htdocs if you are using a localhost server.

The BASE_URL of this project can be found into the folder config file config.php.

The main url is http://127.0.0.1/claromentis

I worked on my localhost using Xampp as a local server, you do not need install any third party packages,
all that the system needs is configured to be used.
